# 👟 JordanShoes

Uma landing page de e-commerce de alta performance visual, focada na linha de sneakers **Air Jordan**. O projeto foi desenvolvido para consolidar conceitos avançados de estilização CSS e estruturação semântica.

## 🚀 Tecnologias Utilizadas

Este projeto foi construído utilizando as seguintes ferramentas:

- **HTML5**: Estruturação semântica utilizando as tags `header`, `main` e `section` para melhor acessibilidade e SEO.
- **CSS3**: Estilização avançada com foco em:
  - **CSS Grid Layout**: Utilizado na seção de produtos com `grid-template-columns: repeat(auto-fit, minmax(300px, 1fr))` para garantir uma malha fluida e responsiva sem necessidade de excessivas media queries.
  - **Flexbox**: Aplicado para o alinhamento de componentes internos, como o cabeçalho e os detalhes dos cards de produtos.
  - **Unidades Relativas (rem)**: Implementação de tipografia e espaçamentos baseados em `62.5%` no `html`, facilitando a escalabilidade e acessibilidade (1rem = 10px).
- **Google Fonts**: Integração da fonte **Archivo** com múltiplas variações de peso (100 a 900).
- **Arquitetura de Arquivos**: Organização modular de estilos separando `reset.css`, `header.css` e `main.css`.

## 🛠️ Funcionalidades e Diferenciais Técnicos

- **Layout Adaptativo**: O projeto utiliza técnicas de design responsivo para ajustar a posição da `background-image` do hero e a quantidade de colunas do grid conforme o viewport.
- **Hero Section Dinâmica**: Uso de `background-size: cover` e `background-position` estratégico para manter o foco visual em telas menores.
- **Cards de Produto**: Componentização visual dos itens com efeitos de sombra (`box-shadow`) e agrupamento lógico de informações (imagem, descrição e preço).
- **Reset de Estilos**: Aplicação de `box-sizing: border-box` e limpeza de margens nativas para garantir consistência entre diferentes navegadores.

## 🎨 Design e Identidade Visual

O layout segue uma estética clean e premium, característica da marca Jordan:

### 🎨 Paleta de Cores

- `#F3F7FF`: Fundo suave para destacar os produtos.
- `#121214`: Tipografia principal para alto contraste.
- `#7CA2F4`: Cor de destaque para categorias e links.

### ✍️ Tipografia

- Uso da fonte **Archivo** para transmitir uma imagem moderna e esportiva.

### 🧩 Componentes

- Cards com padding interno de `5rem` e bordas levemente arredondadas para um acabamento moderno.
